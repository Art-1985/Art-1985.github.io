STM32G474RE REG.

STM32 HAL USED

  1. MX_I2Cx_Init

  2. HAL_I2C_Mem_Write

    3. 簡述 : 以阻塞模式將一定量的數據寫入特定的內存地址
    參數*hi2c : 指向 I2C_HandleTypeDef 結構的指針
    參數DevAddress : 目標設備地址,調用前須將數據的設備7位地址值左移
    參數MemAddress : 內存地址
    參數MemAddSize : 內存地址大小
    參數pData : 指向數據緩衝區的指針
    參數Size : 要發送的數據量
    參數Timeout : 超時時間
    回傳 : HAL status

  3. HAL_I2C_Mem_Read

    4. 簡述 : 從特定內存地址以阻塞模式讀取一定量的數據
    參數*hi2c : 指向 I2C_HandleTypeDef 結構的指針
    參數DevAddress : 目標設備地址,調用前須將數據的設備7位地址值左移

    參數MemAddSize : 內存地址大小
    參數pData : 指向數據緩衝區的指針
    參數Size : 要發送的數據量
    參數Timeout : 超時時間
    回傳 : HAL status

  4. HAL_I2C_Mem_Write_IT

    5. 簡述 : 使用中斷的非阻塞模式將一定量的數據寫入特定的內存地址
    參數*hi2c : 指向 I2C_HandleTypeDef 結構的指針
    參數DevAddress : 目標設備地址,調用前須將數據的設備7位地址值左移
    參數MemAddress : 內存地址
    參數MemAddSize : 內存地址大小

    參數Size : 要發送的數據量
    參數Timeout : 超時時間
